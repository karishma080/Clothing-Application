# 👗 Clothing Application

A modern Clothing Application designed with a user-centered UI/UX approach and developed using React + Vite. This project focuses on creating a clean, responsive, and intuitive shopping experience while showcasing frontend development skills.

---

## 🌟 Overview

This project demonstrates my complete design-to-development workflow.

As a UI/UX Designer, I designed the interface with usability and accessibility in mind and then converted the design into a fully responsive frontend using modern web technologies.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/karishma080/Clothing-Application.git
```

### Navigate to the Project

```bash
cd Clothing-Application
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
